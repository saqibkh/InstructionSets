Add (Shifted Register)

ADD , , {, #}

Adds a register value and an optionally-shifted register value.

Pseudocode Operation

Xd = Xn + Shift(Xm, shift, amount);

Example

ADD X0, X1, X2, LSL #2

// X0 = X1 + (X2 << 2)
