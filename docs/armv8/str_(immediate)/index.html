STR (immediate)

Store Register (Immediate offset)

STR , [, #]

Stores a word or doubleword to memory.

Pseudocode Operation

Mem[Xn + simm] = Xt;

Example

STR X0, [SP, #8]

// Store X0 to StackPointer + 8.
