STR (immediate)

Store Register (Immediate offset)

STR , [, #]

Stores a word or doubleword to memory.

Pseudocode Operation

Mem[Xn + simm] = Xt;

Example

STR X0, [SP, #8]

// Store X0 to StackPointer + 8.

Encoding

Format Load/Store
Opcode 0xF9
Binary Pattern
1x 111001 00 imm12 Rn Rt

Operands

  • Xt Source Reg
  • Xn Base Addr
  • simm Offset
