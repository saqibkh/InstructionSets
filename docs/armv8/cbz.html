CBZ

Compare and Branch on Zero

CBZ ,

Branches if the register is zero.

Pseudocode Operation

if (Xt == 0) PC = PC + offset;

Example

CBZ X0, ErrorHandler

// If X0 is 0, jump to ErrorHandler.
