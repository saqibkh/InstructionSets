LDR (register)

Load Register (Register offset)

LDR , [, ]

Loads a word or doubleword from memory.

Pseudocode Operation

Xt = Mem[Xn + Xm];

Example

LDR X0, [X1, X2]

// Load from address X1+X2 into X0.

Encoding

Format Load/Store
Opcode 0xB8
Binary Pattern
1x 111000 011 Rm 011 S 10 Rn Rt

Operands

  • Xt Target Reg
  • Xn Base Addr
  • Xm Offset Reg
