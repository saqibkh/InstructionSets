C.SRLI

Compressed Shift Right Logical Immediate

C.SRLI rd', imm

Logically shifts a register right by immediate.

Pseudocode Operation

R[rd'] = R[rd'] >> imm;

Example

C.SRLI x8, 2

// x8 = x8 >> 2
