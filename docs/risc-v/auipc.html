AUIPC

Add Upper Immediate to PC

AUIPC rd, imm

Adds a 20-bit upper immediate to the Program Counter.

Pseudocode Operation

R[rd] = PC + (imm << 12);

Example

AUIPC x10, 0

// Loads the current PC value into x10.

Encoding

Format U-Type
Opcode 0x17
Binary Pattern
imm[31:12] | rd | 0010111

Operands

  • rd Dest
  • imm 20-bit offset
