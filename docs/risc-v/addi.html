ADDI

Add Immediate

ADDI rd, rs1, imm

Adds a register and a sign-extended 12-bit immediate value.

Pseudocode Operation

R[rd] = R[rs1] + sext(imm);

Example

ADDI x5, x0, 10

// Loads the value 10 into register x5.
