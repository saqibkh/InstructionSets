JAL

Jump and Link

JAL rd, offset

Jumps to an offset and saves return address.

Pseudocode Operation

R[rd] = PC + 4; PC += sext(offset);

Example

JAL x1, 1024

// Calls function at PC+1024, saves return address in RA (x1).
