FSGNJN.S

Float Sign Injection Negate (Single)

FSGNJN.S rd, rs1, rs2

Injects the *negated* sign of rs2 into rs1. Used for negation and absolute value.

Pseudocode Operation

F[rd] = {~F[rs2][31], F[rs1][30:0]};

Example

FSGNJN.S f1, f2, f2

// Negates f2 (f1 = -f2).
