RISC-V / Base

LW

Load Word

LW rd, offset(rs1)

Loads a 32-bit word from memory.

Pseudocode Operation

R[rd] = M[R[rs1] + sext(offset)][31:0];

Example

LW x5, 8(x2)

// Load value from memory at address (StackPointer + 8) into x5.

Encoding

Format I-Type
Opcode 0x03
Binary Pattern
offset | rs1 | 010 | rd | 0000011

Operands

  • rd Dest
  • rs1 Base Address
  • offset Byte Offset
