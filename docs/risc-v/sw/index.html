SW

Store Word

SW rs2, offset(rs1)

Stores a 32-bit word to memory.

Pseudocode Operation

M[R[rs1] + sext(offset)] = R[rs2][31:0];

Example

SW x5, 12(x2)

// Store x5 into stack memory at offset 12.
