BEQ

Branch if Equal

BEQ rs1, rs2, offset

Branches if two registers are equal.

Pseudocode Operation

if (R[rs1] == R[rs2]) PC += sext(offset);

Example

BEQ x5, x6, 100

// Jump 100 bytes forward if x5 equals x6.
