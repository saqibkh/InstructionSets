LUI

Load Upper Immediate

LUI rd, imm

Loads the 20-bit immediate into the upper 20 bits of the register.

Pseudocode Operation

R[rd] = imm << 12;

Example

LUI x5, 0x12345

// Loads 0x12345000 into x5.

Encoding

Format U-Type
Opcode 0x37
Binary Pattern
imm[31:12] | rd | 0110111

Operands

  • rd Dest
  • imm 20-bit Upper Value
