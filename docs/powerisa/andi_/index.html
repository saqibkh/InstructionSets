AND Immediate

andi. RA, RS, UI

Performs a bitwise AND between a register and a 16-bit unsigned immediate (zero-extended). Always updates CR0.

Pseudocode Operation

RA <- (RS) & (0x0000 || UI); CR0 <- Compare(RA, 0)

Example

andi. r3, r4, 0xF

// r3 = r4 & 0xF
