isel

Integer Select

isel RT, RA, RB, BC

Conditionally copies RA or RB to RT based on a CR bit. (Equivalent to C ternary operator 'cond ? a : b').

Pseudocode Operation

if CR[BC]=1 then RT <- (RA) else RT <- (RB)

Example

isel r3, r4, r5, 2

// r3 = (CR.eq) ? r4 : r5
