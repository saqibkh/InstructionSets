Mnemonic Name Format Summary
add Add XO-form Adds the contents of two registers.
addc Add Carrying XO-form Adds two registers and updates the Carry Architecture (CA) bit.
adde Add Extended XO-form Adds two registers plus the current Carry bit.
addi Add Immediate D-form Adds a 16-bit signed immediate value to a register.
addic Add Immediate Carrying D-form Adds an immediate to a register and updates the Carry bit.
addic. Add Immediate Carrying and Record D-form Adds an immediate, updates Carry, and updates Condition Register Field 0 (CR0).
addis Add Immediate Shifted D-form Adds a 16-bit immediate shifted left by 16 bits to a register.
addme Add to Minus One Extended XO-form Adds a register, -1, and the Carry bit.
addze Add to Zero Extended XO-form Adds a register, 0, and the Carry bit.
and AND X-form Performs a bitwise AND between two registers.
andc AND with Complement X-form Performs a bitwise AND between RS and the one's complement of RB.
andi. AND Immediate D-form Performs a bitwise AND between a register and a 16-bit unsigned immediate (zero-extended). Always updates CR0.
andis. AND Immediate Shifted D-form Performs a bitwise AND between a register and a 16-bit immediate shifted left by 16 bits. Always updates CR0.
b Branch I-form Unconditionally branches to a target address relative to the current instruction pointer.
ba Branch Absolute I-form Unconditionally branches to an absolute address.
bc Branch Conditional B-form Branches conditionally based on the Count Register (CTR) and/or a bit in the Condition Register (CR).
bcctr Branch Conditional to Count Register XL-form Branches to the address in the Count Register (CTR) if the condition is met. Used for computed jumps and switch statements.
bclr Branch Conditional to Link Register XL-form Branches to the address in the Link Register (LR) if the condition is met. Used for function returns.
bl Branch and Link I-form Branches to a target address and saves the return address (CIA + 4) in the Link Register (LR). Used for function calls.
bpermd Bit Permute Doubleword X-form Permutes bits from RS based on the index values in RB. Highly optimized for bit shuffling.
cmp Compare X-form Compares two registers as signed integers and records the result in the specified Condition Register Field.
cmpi Compare Immediate D-form Compares a register to a 16-bit signed immediate.
cmpl Compare Logical X-form Compares two registers as unsigned integers.
cmpli Compare Logical Immediate D-form Compares a register to a 16-bit unsigned immediate.
cntlzd Count Leading Zeros Doubleword X-form Counts the number of consecutive 0 bits starting from bit 0 (MSB of 64-bit reg).
cntlzw Count Leading Zeros Word X-form Counts the number of consecutive 0 bits starting from bit 32 (MSB of the low word).
crand Condition Register AND XL-form Performs a bitwise AND between two bits in the Condition Register.
cror Condition Register OR XL-form Performs a bitwise OR between two bits in the Condition Register.
crxor Condition Register XOR XL-form Performs a bitwise XOR between two bits in the Condition Register. Used to clear CR bits (crxor x,x,x).
dadd Decimal Add X-form Adds two 64-bit Decimal Floating Point (DFP) numbers. Used in financial calculations to avoid rounding errors.
dcbf Data Cache Block Flush X-form Flushes the cache block from the data cache to main memory and invalidates it. Used for DMA coherency.
dcbst Data Cache Block Store X-form Writes the cache block to main memory if it is modified (Clean), but keeps it in the cache.
dcbt Data Cache Block Touch X-form Hints to the hardware to prefetch the cache block at the specified address into the cache.
dcbz Data Cache Block Set to Zero X-form Zeros out an entire cache block (usually 128 bytes) in memory. Critical for optimizing memory clears (memset).
divd Divide Doubleword XO-form Divides the 64-bit value in RA by the 64-bit value in RB (Signed).
divdu Divide Doubleword Unsigned XO-form Divides the 64-bit value in RA by the 64-bit value in RB (Unsigned).
divw Divide Word XO-form Divides the lower 32 bits of RA by the lower 32 bits of RB (Signed).
divwu Divide Word Unsigned XO-form Divides the lower 32 bits of RA by the lower 32 bits of RB (Unsigned).
dmul Decimal Multiply X-form Multiplies two 64-bit Decimal Floating Point numbers.
dqua Decimal Quantize X-form Adjusts the exponent of a DFP number to match a reference. Critical for aligning decimal points before addition.
eieio Enforce In-order Execution of I/O X-form Ensures that load/store instructions preceding the EIEIO complete before those following it. Used for Memory-Mapped I/O synchronization.
eqv Equivalent X-form Performs a bitwise Equivalence (XNOR) operation. (RA = ~(RS ^ RB)).
extsb Extend Sign Byte X-form Sign-extends the low byte (8 bits) of a register to 64 bits.
extsh Extend Sign Halfword X-form Sign-extends the low halfword (16 bits) of a register to 64 bits.
extsw Extend Sign Word X-form Sign-extends the low word (32 bits) of a register to 64 bits.
fadd Floating Add A-form Adds two double-precision floating-point registers.
fcmpu Floating Compare Unordered X-form Compares two floating-point registers and sets the Condition Register (CR) field. Does not trap on NaNs.
fctiw Floating Convert to Integer Word X-form Converts a float to a 32-bit signed integer (using the current rounding mode) and stores it in the lower half of the FPR.
fmadd Floating Multiply-Add A-form Performs (A * C) + B with a single rounding step. (The classic FMA).
fmul Floating Multiply A-form Multiplies two double-precision floating-point registers.
icbi Instruction Cache Block Invalidate X-form Invalidates the instruction cache block associated with the address. Critical for self-modifying code or JITs.
isel Integer Select A-form Conditionally copies RA or RB to RT based on a CR bit. (Equivalent to C ternary operator 'cond ? a : b').
isync Instruction Synchronize XL-form Waits for all previous instructions to complete and discards any prefetched instructions. Used after modifying code or changing context.
lbz Load Byte and Zero D-form Loads a byte from memory into the low 8 bits of a register and clears the upper 56 bits.
ld Load Doubleword DS-form Loads a doubleword (64 bits) from memory.
ldarx Load Doubleword and Reserve Indexed X-form Loads a doubleword and creates a reservation. 64-bit version of lwarx.
lha Load Halfword Algebraic D-form Loads a halfword (16 bits) from memory and sign-extends it to 64 bits.
lhz Load Halfword and Zero D-form Loads a halfword (16 bits) from memory and clears the upper 48 bits.
lvx Load Vector Indexed X-form Loads a 128-bit vector from memory into a Vector Register. Address must be 16-byte aligned (bits 60-63 of effective address are ignored).
lwa Load Word Algebraic DS-form Loads a word (32 bits) from memory and sign-extends it to 64 bits.
lwarx Load Word and Reserve Indexed X-form Loads a word and creates a reservation for use with 'stwcx.'. Critical for implementing atomic primitives (mutexes).
lwsync Lightweight Synchronize (Pseudo) Pseudo Orders loads with loads, stores with stores, and loads with stores. Does NOT order stores with loads. (Encoded as sync 1).
lwz Load Word and Zero D-form Loads a word (32 bits) from memory and clears the upper 32 bits.
lxvd2x Load VSX Vector Doubleword*2 Indexed XX1-form Loads a 128-bit vector from memory into a VSX register. Does NOT require 16-byte alignment (unlike lvx).
mcrf Move Condition Register Field XL-form Copies the contents of one Condition Register field to another. Used to save comparison results.
mfcr Move From Condition Register X-form Copies the entire 32-bit Condition Register into a General Purpose Register.
mfspr Move From Special Purpose Register XFX-form Copies a value from a system SPR to a general-purpose register.
mr Move Register (Pseudo) Pseudo Copies the contents of one register to another. (Encoded as OR RA, RS, RS).
mtcrf Move To Condition Register Fields XFX-form Copies bits from a register into the Condition Register, updated only the fields specified by the mask (FXM).
mtspr Move To Special Purpose Register XFX-form Copies a value from a general-purpose register to a system SPR (e.g., CTR, LR, XER).
mulld Multiply Low Doubleword XO-form Multiplies two 64-bit integers and stores the lower 64 bits of the 128-bit result.
mullw Multiply Low Word XO-form Multiplies two 32-bit integers and stores the lower 32 bits of the 64-bit result.
nand NAND X-form Performs a bitwise NAND comparison. (RA <- ~(RS & RB)).
neg Negate XO-form Computes the two's complement negation of a register (0 - RT).
nor NOR X-form Performs a bitwise NOR comparison. (RA <- ~(RS | RB)). Useful for bitwise NOT (nor r3, r4, r4).
or OR X-form Performs a bitwise OR comparison.
orc OR with Complement X-form Performs a bitwise OR between RS and the complement of RB.
ori OR Immediate D-form Performs a bitwise OR with a 16-bit unsigned immediate.
oris OR Immediate Shifted D-form Performs a bitwise OR with a 16-bit immediate shifted left by 16 bits.
paddi Prefixed Add Immediate MLS:D-form (64-bit) Adds a 34-bit signed immediate to a register. Supports PC-relative addressing (LEA).
pla Prefixed Load Address (Pseudo) Pseudo Pseudo-instruction for paddi with R=1. Loads the address of a label.
plbz Prefixed Load Byte and Zero MLS:D-form Loads a byte and zero-extends it, using a 34-bit offset.
pld Prefixed Load Doubleword MLS:D-form Loads a 64-bit value from memory using a 34-bit immediate offset (PC-relative or absolute).
plwz Prefixed Load Word and Zero MLS:D-form Loads a 32-bit word and zero-extends it to 64 bits, using a 34-bit offset.

popcntw Population Count Word X-form Counts the number of set bits (1s) in the lower 32 bits of a register.

rfid Return From Interrupt Doubleword XL-form Returns from an interrupt handler. Restores PC from SRR0 and MSR from SRR1.
rldic Rotate Left Doubleword Immediate Clear MD-form Rotates a 64-bit register left, then clears bits based on a mask. 64-bit equivalent of rlwinm.
rldicl Rotate Left Doubleword Immediate Clear Left MD-form Rotates 64-bit RS left by SH, then clears the high-order bits (0 to MB-1). Often used to extract bitfields.
rldicr Rotate Left Doubleword Immediate Clear Right MD-form Rotates 64-bit RS left by SH, then clears the low-order bits (ME+1 to 63).
rlwimi Rotate Left Word Immediate Then Mask Insert M-form Rotates RS left by SH, then inserts bits into RA under the control of a mask. Used to merge bitfields from two registers.
rlwinm Rotate Left Word Immediate Then AND with Mask M-form The ultimate bit-manipulation instruction. Rotates RS left by SH, then ANDs with a mask defined by MB (Mask Begin) and ME (Mask End).
