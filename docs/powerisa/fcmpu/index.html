fcmpu

Floating Compare Unordered

fcmpu BF, FRA, FRB

Compares two floating-point registers and sets the Condition Register (CR) field. Does not trap on NaNs.

Pseudocode Operation

if NaN(FRA) or NaN(FRB) then CR[BF] <- U; else compare...

Example

fcmpu cr0, f1, f2

// Compare f1 vs f2.
