LEA

Load Effective Address

LEA r64, m

Computes effective address and stores it in register.

Pseudocode Operation

DEST = EffectiveAddress(SRC);

Example

LEA RAX, [RBX+8]

// Calculates address RBX+8 and puts it in RAX (does not access memory).

Encoding

Format CISC Variable
Opcode 8D
Binary Pattern
REX.W + 8D /r

Operands

  • dest Dest Register
  • src Memory Address Calculation
